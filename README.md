## Installation

### Bash version

1. Rename the script:

```bash
mv genf.sh genf
```

2. Make it executable:

```bash
chmod +x genf
```

3. Install it:

```bash
sudo mv genf /usr/local/bin/
```

### C version

1. Compile the source:

```bash
gcc -O2 -o genf genf.c
```

2. Install the compiled binary:

```bash
sudo mv genf /usr/local/bin/
```
